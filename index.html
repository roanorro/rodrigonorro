Configuración requerida
Necesitás una cuenta gratuita en jsonbin.io para guardar scripts en la nube.
  1. Creá cuenta gratis en jsonbin.io
  2. En el panel, hacé clic en + Create Bin y poné {} como contenido
  3. Copiá el BIN ID que aparece en la URL del bin (es el código largo)
  4. Andá a tu perfil → API Keys → copiá tu Access Key
  5. Abrí este archivo HTML con un editor de texto y reemplazá estas dos líneas:
const JSONBIN_ID  = 'PEGAR_BIN_ID_ACÁ';
const JSONBIN_KEY = 'PEGAR_API_KEY_ACÁ';
Subí el archivo a Netlify Drop (arrastrar y soltar) y obtenés una URL fija gratis.
Esa URL es la que vas a abrir siempre, desde casa y del trabajo.
